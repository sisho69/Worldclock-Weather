# Worldclock-Weather

世界各地の時間と天気をリアルタイムで表示するアプリケーションです。

## 機能

- 複数都市の時刻を同時表示
- 時間帯に応じたグラデーション背景
- OpenWeatherMap API を使用した天気表示


## 作業時間
- 約2か月

## 技術スタック

- React
- TypeScript
- Vite

## セットアップ

```bash
npm install
npm run dev
```
